Getting started

Installing
路 Clone this repo git clone https://github.com/saerose/minerva-client
路 Move into the new folder cd minerva-client
路 Install all dependencies npm install
路 To start the front-end run npm start. This activates the react start script and will display the front-end, you will need to have the back-end live in order to test out functionality.[ https://github.com/saerose/minerva-server ]
